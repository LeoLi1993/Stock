## 个股如何做T
1.适合场景：个股上涨回调企稳的时候才能够去做T，或者上涨趋势中去做低吸高抛，分时图上快速的之前拉升（分时图二次无量拉升，第二个波段比第一个小很多，拐头的时候我们就扔了），一口气超过5个点
1.1 一般早盘逆势拉升，可以去做高抛，一般这种是主力自救或者是建仓的行为
1.2 盘中个股逆势直线拉升，可以高抛
1.3 必须是市场热点版块的个股才可以做T，一定要交易活跃，在多头趋势中的回调企稳做T
1.4 大盘稳定，但是个股早盘杀跌，个股并且在支撑位或者相对低位的多头趋势中，早盘可以低吸，这种是洗盘。
2.不适合场景：个股已经开始杀跌，那么我们应该离场而不是高抛

## 主升浪的控盘线
1.控盘线只在上涨趋势中有效，是上涨回调的支撑位，怎么画见截图
<img width="2355" height="1210" alt="image" src="https://github.com/user-attachments/assets/c7f83d89-ce39-4d2a-a6e6-dd58efcee9b6" />
2. 主升浪的票上涨斜率超过45度的，用5日和10日做趋势线，否则用修正趋势线来控制
3.大盘票主升浪往往是3浪，三浪的1/2往往是支撑，也就是2浪的低点
4.如果回落空间比较小，那么强势股按照10日均线为限制条件，10日线出现放量的阳包阴或者十字星加阳线是加仓点

5.加速
-    高位缩量大阳线或者缩量一字板
-    放量加速：分时盘口上迟迟不封板，放大量
-    加速之后容易出现A杀

## 主升浪的洗盘
1.主升浪初期一定是控盘良好，不会出现大的波动和震荡

``` 主力控盘这个因素特别的需要注意，控盘后，代表仓位比较重，代表上涨空间比较大
主力进入主升浪，基本上都有一个目标，洗盘一般发生在两种情况下，一种到了目标位，30一50个点位，一种遇到了大的抛压(一字板)
@1、假阴线:就是洗盘的时候保留一定的缺口。
@2、阴覆盖或者应贯穿:(第二根线型的佐证
@3、分歧线型:高浪线，流星线，吊颈线，共同的特点:
1、承接比较高，不会形成特别恶劣线型(穿头破脚，破均线)注明:主力没有出货的情况下，一般不会形成破位技术结构。
2、过度振荡线型，地天板等，(渝开发)
3、分歧时间比较短。
技术破坏要点:
@1、小盘子:10均线的破坏。
@2、趋势线的破坏离场。(原始趋势线，修正趋势线)
你不要去轻易的去看空
```


## 主升浪强势结构
1. 强势突破的要求:
@强势突破:要求的阳线是涨停板或者是超过6个点以上的大阳线，温和放量(杜绝爆量)
@重点在第二根线型，第二根应该继续上攻收阳，量能均衡(一字板除外)

```

1、小盘子票，第一个根是涨停板，上午封板，(尽可能不开板)，第二根阳线和第一根保留缺口(起涨缺口)
操作:错过第一个，第二阳线是扫板为主，及可能不要去低吸(确定性)两根阳线奠定主升的基础。形成趋势的情下，第三根线型低吸为主(5日均线）
如果是高开低走的你的**极限10均线**。
均线支撑位5日均线，斜率超过60度以上。
均线的使用方法:主升浪之前均线会黏连，5、10、20、30日，60均线呈现向上的多头发散。

2、中大盘票:第一个根是大阳线，是否涨停无所谓(破板或者尾盘破板无所谓。)第二阳线平开或者略微低开都可以，低吸，10均线的支撑，斜率小于45度。
共同特征:
第二根阳线继续上政，为回调制造空间，主力也是迅速拉离成本区。区别于假突破，第二根线型是回调，或者高低走的阴线，重点看的颈线为主是否能守住，受不住就是假突破，线下止损.
第三根阳线:红三兵技术特征，确立形成主升浪。
操作要点:在支撑位附近逐步的把仓位加大。
补充一下N型结构(多方炮)
第二，或者第三根出现了回调，技术条件不破颈线位置，然后(在支撑位之上)再次出现大阳线，这种技术形态是典型看多的技术结构，后面是走主升浪。```
